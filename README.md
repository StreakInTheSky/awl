# Art with Larisse

> A Vue.js project using VueRouter and Firebase API

## Build Setup

``` bash
# install dependencies
npm install -g yarn
yarn

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

## Component Options

``` javascript
// Hero Component options
options = {
  img_name: String,
  pre_title: String,
  title: String,
  link_text: String,
  link_path: String
}
```
